# Salaam Afghanistan — Public Website

Next.js (App Router) + TypeScript applicant-facing site.

## Quick start

```bash
cd website
cp .env.example .env.local
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home — hero apply form, destinations, platform intro, opportunities, about preview, process, FAQ |
| `/about` | About — story, values, opportunities, CTA |
